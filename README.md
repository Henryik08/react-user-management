# React User Management

A simple React app for managing users with TypeScript, Redux Toolkit, and React Router.

## What I learned

- How to use Redux Toolkit to keep state logic clean and predictable.
- How to fetch and manage async data with thunks.
- How to wire up multiple routes with React Router.

## Tech stack

- React
- TypeScript
- Vite
- Redux Toolkit
- React Router DOM

## Installation

1. Clone the repo
2. Run `npm install`
3. Start the app with `npm run dev`

## Available routes

- `/users` — user list
- `/users/:id` — user details
- `/add-user` — add a new user
- `/edit-user/:id` — edit an existing user
